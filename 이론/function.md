# 함수
- 특정한 작업을 수행하도록 제작된 독립적인 코드
- 코드덩어리 + 이름 = 함수

# 함수를 사용하는 이유
- 편하다
- 코드의 재사용이 가능하다 !!
- 전체 구조를 파악하기 쉽다

# 함수의 종류

- 제공 함수(표준 함수)
- 제작 함수

# 함수의 기본 형태
리턴형 함수명 (데이터형 변수, 데이터형 변수){
    함수 본문 : '특정한 작업을 수행하도록 설계된 코드'
    return data; 리턴형에 따라 생략 가능(ex: void)
}

# 제공 함수 예시
 알파벳 대문자를 소문자로 변환

```c
#include <stdio.h>
#include <ctype.h>

void main(){
    int cMun = 'A';
    cMun = tolower(cMun);
    printf("%c \n, cMun);
    return 0;
}
```
